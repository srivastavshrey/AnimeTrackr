
# AnimeTrackr

A web app to track anime you're currently watching and explore new anime.


## Installation

Clone the repo:

```bash
git clone https://github.com/srivastavshrey/anime-tracker
```
Open repo in code editor, then:
```bash
npm install
or
yarn install
```
To run the project:
```bash
1.npm run server
2.npm start or yarn start
